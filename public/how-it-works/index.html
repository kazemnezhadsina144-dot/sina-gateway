How it works

One intake form. Live sorting to the right product line. A confirmation code on screen — not a black-hole form.

Founder Audit signal path (example)

  1. You arrive from Founder Audit or ?utm_campaign=founder-audit.
  2. You complete the wizard: who you are → goal → value → timeline → contact.
  3. The routing console shows the likely product line before you submit.
  4. On submit you receive a confirmation code and which product line will review.
  5. Sina reviews within 48 business hours using the contact you provided.

No email confirmation unless explicitly added later. Urgent inquiries can alert @Gateway_A.
